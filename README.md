# TGames – Landing Page (Тестовое задание)

Реализация главной страницы проекта на **Nuxt 4** в соответствии с техническим заданием.

[ПОСМОТРЕТЬ ДЕМО КЕЙСА](https://shatobiaan.github.io/nuxt4-test-task/)

## Стек технологий

| Инструмент | Назначение |
|---|---|
| **Nuxt 4** + Vue 3 | Основной фреймворк (SSR) |
| **Tailwind CSS v3** | Стилизация (без кастомного CSS) |
| **Swiper** | Слайдер баннеров |
| **Zod** | Валидация формы |
| **vue-sonner** | Toast-уведомления |
| **@vueuse/core** | `onClickOutside` для модала |
| **nuxt-schema-org** | Структурированные данные (SEO) |

## Запуск проекта

1. **Настройте окружение**:
   Скопируйте пример файла конфигурации и при необходимости подправьте значения:
   ```bash
   cp .env.example .env
   ```

2. **Установите зависимости**:
   ```bash
   pnpm install
   ```

3. **Запустите в режиме разработки**:
   ```bash
   pnpm dev
   ```

4. **Сборка продакшн**:
   ```bash
   pnpm build
   ```

Приложение будет доступно на `http://localhost:3000`

## Реализованные требования

### Обязательное
- ✅ **SEO мета-данные** — динамические `title` и `description` из API (`useSeoMeta`)
- ✅ **Слайдер баннеров** — Swiper с автоплеем, пагинацией и навигацией
- ✅ **Сетка товаров** — адаптивный grid (1 → 2 → 3 → 4 колонки)
- ✅ **Карточка товара** — фото, название, цена, кнопка «Заказать»
- ✅ **Попап с формой** — открывается по клику на «Заказать»
- ✅ **Поля формы** — название товара (readonly), ФИО, номер телефона
- ✅ **Отправка формы** — данные собираются, выводятся в `console.log` (согласно ТЗ) и показывается toast-уведомление

### Плюсы (реализованы)
- ✅ **Loading state** — Skeleton-лоадеры, повторяющие форму слайдера и карточек
- ✅ **Error state** — Экран с ошибкой и кнопкой «Перезагрузить»
- ✅ **Empty state** — «По вашему запросу ничего не найдено» при поиске
- ✅ **Структура проекта** — composables, atomic UI-компоненты, строгая типизация
- ✅ **Валидация формы** — Zod: минимум 2 слова в ФИО, только буквы, минимум 11 цифр в телефоне
- ✅ **Чистый код** — только Tailwind-классы, без кастомного CSS

### Дополнительно (сверх ТЗ)
- Маска ввода телефона `+7 (___) ___-__-__`
- Живой поиск по товарам с анимацией карточек
- Schema.org разметка (`WebPage`, `WebSite`, `Product`)
- `lang="ru"` для правильной SEO-индексации

```json
      "ELEMENT_META_DESCRIPTION": "..."
    }
  }
}
```
